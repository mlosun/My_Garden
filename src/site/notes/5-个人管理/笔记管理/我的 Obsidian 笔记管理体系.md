## 🗂️ 笔记分类方法

基于自身需求和关注点的笔记分类方法：

| 文件夹 | 用途说明 |
|--------|----------|
| `1-工作笔记/` | 根据工作需求调整内部结构 |
| `2-项目管理/` | 工作之余所关注的一些项目 |
| `3-每日笔记/` | 日/周/月/年笔记，按年组织 |
| `4-数字花园/` | 公开分享的内容，同步到网络 |
| `5-个人管理/` | 个人资料、财务、家庭等内容 |
| `6-仓库内务/` | 附件、模板、组件、归档等 |

## 🧩 笔记属性系统

### 基础属性（必需）

每篇笔记至少包含以下属性（使用 Templater 和 Linter 插件自动维护）：

```yaml
---
created: 2026-01-31      # 创建日期
updated: 2026-01-31      # 更新日期（Linter 自动维护）
type: Daily              # 笔记类型
status: 🌱 活跃          # 笔记状态
---
```

### 扩展属性（可选）

```yaml
---
rating: ⭐️⭐️⭐️⭐️⭐️     # 内容质量评分（1-5）
dg-publish: false        # 是否发布到数字花园（数字花园插件）
---
```

### Type 类型清单

| 类型 | 说明 |
|------|----------|
| `README` | 目录索引笔记 |
| `Daily` | 每日笔记 |
| `Weekly` | 周度笔记 |
| `Monthly` | 月度笔记 |
| `Yearly` | 年度笔记 |
| `Reviews` | 回顾笔记 |
| `Zettelkasten` | 卡片笔记 |
| `Evergreen` | 常青笔记 |
| `Reference` | 参考笔记 |
| `Learning` | 学习笔记 |
| `Notes` | 默认笔记 |

### Status 状态清单

| 状态         | 说明         |
| ---------- | ---------- |
| 📥 新建  | 尚未定义的笔记    |
| 🌱 活跃  | 当前高频使用的笔记  |
| ⭐️ 星标  | 长期高频使用     |
| 🕊️ 搁置 | 将来也许，未完成   |
| ⌛️ 等待  | 等待特定时间处理   |
| ✅ 完成   | 处理后的笔记，已发布 |
| 📦 归档  | 归档后的笔记     |

## ⌨️ 常用快捷键

| 快捷键 | 功能说明 |
|--------|----------|
| `Cmd + 1` | 打开今天的日记 |
| `Cmd + 2` | 打开添加记录表单 |
| `Cmd + 3` | 创建或修改任务 |
| `Cmd + S` | 触发 Linter 格式化（编辑模式） |

## 📚 附录：参考框架

### ACCESS 框架

| 字母 | 含义 | 说明 |
|------|------|----------|
| A | Atlas | 地图集，MOC 导航 |
| C | Calendar | 日历，时间相关笔记 |

| E | Extras | 附属，模板和脚本 |
| S | Sources | 资源，参考资料 |

| CODE | Capture, Organize, Distill, Express | 捕获、组织、提炼、表达 |
